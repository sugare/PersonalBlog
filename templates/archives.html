    Linux Blog is a free template from Sumanasa.com released under a Creative Commons License. This is a simple linux-themed template with a header, a horizontal menu, content and sidebar columns. We have tested this template in Firefox 1.5, Opera 8.5, IE 6.0 and IE 7.0 and it works fine in all these browsers. This is a valid XHTML and CSS and it is also very easy to customize.
